## 文件说明

### 后端分层职责

| 层 | 目录 | 职责 |
|----|------|------|
| Handler | `handler/` | 接收 HTTP 请求，参数校验，调用 Service，返回响应 |
| Service | `service/` | 业务逻辑，组合调用 Model 和 K8s 操作 |
| Model | `model/` | 数据库 CRUD 操作 |
| K8s | `k8s/` | K8s API 调用封装 |
| Middleware | `middleware/` | 请求拦截（认证、审计、权限） |
| Pkg | `pkg/` | 通用工具，无业务依赖 |

### 关键依赖包

| 包 | 用途 |
|----|------|
| `github.com/gin-gonic/gin` | HTTP 框架 |
| `gorm.io/gorm` + `gorm.io/driver/mysql` | ORM |
| `github.com/golang-jwt/jwt/v5` | JWT |
| `golang.org/x/crypto/bcrypt` | 密码加密 |
| `k8s.io/client-go` | K8s API 客户端 |
