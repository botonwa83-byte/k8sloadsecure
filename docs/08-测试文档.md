## 1. 项目功能概述

### 1.1 用户管理模块
- 用户创建、编辑、删除
- 密码重置与过期管理
- 三角色体系：admin、global_viewer、developer

### 1.2 项目管理模块
- 项目创建与命名空间关联
- 用户-项目权限分配
- 默认只读权限配置

### 1.3 K8s Dashboard 集成
- ServiceAccount 自动创建
- 基于角色的 RBAC 权限同步
- Dashboard 代理访问

### 1.4 审计日志模块
- 操作日志记录与查询
- 登录日志追踪
- 统计分析接口

### 1.5 权限审批模块
- 开发者写权限申请
- 管理员审批机制
- 24小时权限自动过期
